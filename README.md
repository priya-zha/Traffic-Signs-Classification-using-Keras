# 🚦 Traffic Signs Classification using Keras

A deep learning project that classifies traffic sign images into their respective categories using a Convolutional Neural Network (CNN) built with Keras. This project is relevant to autonomous driving and road safety systems.

---

## 📌 Description

This project trains a CNN to recognize and classify various traffic signs from image data. Traffic sign recognition is a critical component in self-driving vehicles and driver-assistance systems. The model is trained on a labeled dataset of traffic sign images and learns to distinguish between multiple sign categories with high accuracy.

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/priya-zha/traffic_signs_classification_keras.git
cd traffic_signs_classification_keras
```

### 2. Install Dependencies
```bash
pip install tensorflow keras numpy matplotlib pandas jupyter
```

### 3. Download the Dataset
- The project uses the **German Traffic Sign Recognition Benchmark (GTSRB)** dataset or similar
- Download and place it in the appropriate directory as referenced in the notebook

### 4. Launch the Notebook
```bash
jupyter notebook traffic_signals_classification_keras.ipynb
```

---

## 🚀 Usage

1. Open the notebook in Jupyter or Google Colab
2. Run cells in order to:
   - Load and preprocess traffic sign images
   - Apply data augmentation techniques
   - Build and compile the CNN architecture
   - Train the model on the dataset
   - Evaluate model accuracy and visualize predictions
